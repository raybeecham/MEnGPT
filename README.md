# MEnGPT Service Starter (FastAPI)

Production-ready FastAPI starter with secure defaults, CI checks, and tests.

## Tech Stack

- Python 3.11+
- FastAPI + Uvicorn
- Pytest
- Ruff

## Setup (Windows/WSL + VS Code)

```bash
python -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install -e .[dev]
cp .env.example .env
```
